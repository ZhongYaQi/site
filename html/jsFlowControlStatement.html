条件判断语句 | if语句

对条件表达式进行求值判断，根据布尔值结果执行不同语句

语句和语句间可嵌套
一个代码块被执行后结束整条语句

开发中尽量写代码块，使结构更清晰


if(条件表达式){
 语句
}

if(条件表达式){
 语句
}else{
 语句
}

if(条件表达式){
 语句
}else if(条件表达式){
 语句
}else{
 语句
}





条件分支语句 | switch语句

执行时依次将case后的值和条件表达式的值全等比较
值为true从当前case处开始执行代码
值为false继续向下比较
break退出switch语句
default默认执行位置，所有结果为false时执行

{case 值: 语句 break;}可多次使用


switch(条件表达式){
 case 值:
   语句
   break;
 default:
   语句
   break;
}





循环语句 | while语句

反复执行一段代码
while里代码块称为循环体

创建循环的3种构成要素:
①初始化变量
var i=0;
②在循环中设置条件表达式
while(条件表达式)
③定义更新表达式
每循环一次更新一次初始变量
i++;



while循环 先判断后执行
判断条件表达式，值为true执行循环体，直至值为false终止循环


var i=0;
while(条件表达式){
 语句
 i++;
}

do..while循环 先执行后判断
执行循环体后对条件表达式进行判断
值为true继续执行循环体
值为false终止循环

循环体在判断前会被执行1次


var i=0;
do{
 语句
 i++;
}while(条件表达式)





循环语句 | for循环

循环中的三个部分可以写在外部
*分号不能省略

语法:
for(①初始化表达式; ②条件表达式; ④更新表达式){
 ③语句
}

执行流程:
①初始化表达式(只会执行一次)
②条件表达式判断是否执行循环
③true执行循环体，false终止循环
④执行更新表达式，继续判断条件表达式


for(var i=0; 条件表达式; i++){
 语句
}
