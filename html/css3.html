transform 2D或3D转换

transform:平移 旋转;

可添加多个属性值，注意旋转应位于最后位，旋转会改变坐标系，优先旋转会改变后位属性值的方向

效果执行完后默认还原状态


2D example 3D example

translate() 平移 2D
translateX() 水平方向平移
translateY() 垂直方向平移
参数 X轴方向平移数值
参数，参数 X轴平移数值，y轴平移数值

scale() 缩放几倍 2D
从中心缩放
scaleX() 水平方向缩放
scaleY() 垂直方向缩放

(参数) 表示X和Y方向
(参数，参数) 表示X方向，Y方向
1 默认，不缩放
数值<=0.99 缩小
数值>=1.01 放大

rotate() 旋转 2D
以中心点为定点顺时针旋转
旋转会将当前坐标系旋转
0deg 默认0度

skew() 斜切 2D
skewX() 水平方向斜切
skewY() 垂直方向斜切

正值deg，向反方向斜切(左上)
-负值deg，向正方向斜切(右下)
参数 在X轴上斜切
参数，参数 在X轴上斜切，在Y轴斜切

translate3d() 移动 3D
translateX()
translateY()
translateZ()
X,Y,Z 参照原始坐标轴偏移

Z轴在三维图形内移动


scale3d() 缩放 3D
scaleX()
scaleY()
scaleZ()
1 默认，不缩放
数值<=0.99 缩小
数值>=1.01 放大

Z轴缩放关联厚度


rotate3d() 旋转 3D
X,Y,Z,角度 X轴上的向量值，Y轴上的向量值，Z轴上的向量值，角度

属性值给出的向量值代表一个方向，表示从圆心点在相应的轴上拉出到属性值的距离，数值差异不影响实现
(1,0,0,60deg)，在X轴上向上翻转
(0,1,0,60deg)，在Y轴上向右翻转
(1,1,0,60deg)，平分X和Y轴间的角度，在45deg轴上向右上翻转
(1,10,0,60deg)，Y轴上的向量值占比更多，翻转时以Y轴偏移角度最小的轴为主轴，向右上翻转
(10,1,0,60deg)，主轴靠近X轴，向上右翻转
(1,1,1,60deg)，绕三维图形内部对角线翻转

角度：正值仰视，负值俯视


transform-style:; 3D 容器属性 保留3D转换后的状态
flat 默认，不保留
preserve-3d 保留

perspective:; 3D 容器属性 透视距离
值小于三维图形底部的一半 视角在三维图形内部
0，无效值，位于三维图形中间的切面
1px，back面呈现为点状
值等于三维图形底部的一半 视角在front面
值大于三维图形底部的一半 视角从内部移出至外部
值越大，back面越拉近，直至三维图形和视角的原始状态

perspective-origin:; 3D 容器属性 镜头视角的位置
0 0 图形前后面正对屏幕
X轴 Y轴 负值，图形在轴上分别向右和向下转
正值，图形在轴上分别向左和向上转

transform-origin:; 自定义轴心点
关键字 left top
x y X轴 Y轴

transition 过渡

transition:属性名称 过渡时长 时间函数 延迟显示;

指定多个过渡属性，多个简写集用逗号分隔

过渡效果执行完后默认还原状态

写在需过渡的元素样式下


transition-property:; 应用过渡效果的属性名称
只能作用于给出具体值的属性
none 没有属性有过渡效果
all 所有属性都有过渡效果
不建议使用遍历所有属性，至效率低下
指定的属性 多个属性以逗号分隔

transition-duration:; 过渡的时长

 默认值0，单位为秒(s)或毫秒(ms)


transition-timing-function:; 设置时间函数，控制运动中的速度
linear 匀速
ease 默认，慢快慢
ease-in 以慢速开始
ease-out 以慢速结束
ease-in-out 以慢速开始和结束
steps() 跳跃状
步长值，分为几次来完成

transition-delay:; 延迟过渡效果





animation & @keyframes 动画


animation:; 添加动画效果


animation example

animation-name:; 定义@keyframes的动画名

animation-duration:; 动画时长

animation-iteration-count:; 动画播放次数
1 默认值
infinite 无限的

animation-direction:; 动画播放方向
alternate 执行完动画后，按路径返回

animation-delay:; 动画延迟

animation-fill-mode:; 动画执行状态
forwards 保留动画执行后的状态
backwards (设有动画延迟)起始状态为执行transform的状态
both 既保留结束状态(forwards)也初始状态(backwards)

animation-timing-function:; 动画的时间函数，控制运动中的速度
linear 匀速
ease 默认，慢快慢
ease-in 以慢速开始
ease-out 以慢速结束
ease-in-out 以慢速开始和结束
steps() 跳跃状
步长值，分为几次来完成

animation-play-state:; 暂停播放
paused 暂停
running 运行

<input type="button" value="播放" id="running">
<input type="button" value="暂停" id="paused">
<script>
 var 动画容器=document.querySelector("动画容器");
 document.querySelector("running").onclick=function(){
  动画容器.style.animationPlayState="running";
 }
 document.querySelector("paused").onclick=function(){
  动画容器.style.animationPlayState="paused";
 }
</script>
