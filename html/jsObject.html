Function

声明形参(形式参数)，等于在函数内部声明了对应的未赋值的变量
指定实参(实际参数)，赋值给函数中对应的形参
实参可以是任意数据类型

解析器不会检查实参的类型
*有风险接收到非法参数，需对参数进行类型检查

解析器不会检查实参的数量
实参多于形参，多出的实参不会被赋值给形参
实参少于形参，没有对应实参的形参值为undefined


创建函数 (使用函数声明)
同时声明形参
function 函数名(形参){语句} 语法


创建函数 (使用函数表达式)
同时声明形参
将匿名函数赋值给变量
赋的值为函数对象，赋值语句以分号结束
var 函数名=function(形参){语句}; 语法


创建函数 (调用构造函数)
以字符串的形式封装功能代码
var 函数名=new Function("功能代码"); 语法


调用函数
同时指定实参
函数对象(实参); 语法

调用时执行函数
调用的实参非函数对象时，意味着实参就是返回值
调用的实参是函数对象时，意味着在直接使用函数对象
可以在调用的函数对象中定义该函数对象的实参
函数对象(函数对象2(实参)); 语法

开发中常用到将匿名函数作为实参传递给函数对象
函数对象(function(){语句}); 语句


返回值
将函数返回至调用函数
返回的值可以是任意类型，比如运算语句
不设值和不写该语句，输出的调用函数类型为undefined
return执行后终止函数执行

设置返回值至调用函数，可以在函数外部处理输出功能的形式
return 值; 语法
var 执行结果=函数对象(实参); 定义变量接收函数的执行结果

构造函数
一个构造函数称为一个类
使用同一个构造函数创建的对象，称为一类对象
通过一个构造函数创建的对象，称为该类的实例

构造函数执行流程
①调用构造函数，立刻在堆内存中创建空间新建一个新的对象
②将函数自主创建的对象设置为函数中的this，在构造函数中使用this来引用函数自主创建的对象
③逐行执行函数中的代码
④将函数自主创建的对象作为返回值返回


创建构造函数
本质是普通函数，用于创建对象
通常但不强制函数名首字母大写
function 函数名首字母大写(){} 语法


调用构造函数
使用new关键字调用的函数声明创建的函数就是构造函数
new 函数名(); 语法


instanceof 检查一个对象是否是一个类的实例
返回布尔值
所有对象都由Object衍生，所以和Object做检查都会返回true
对象 instanceof 构造函数 语法
