String 字符串

基本数据类型


"" '' 字符需要被引用
发生嵌套时，可使用转义字符转义 \" \'
换行 \n
制表符(Tab键) \t
反斜杠 \\
Unicode的编码 \u
转Number 空字符串=0
不是数字的字符串=NaN
转Boolean 空字符串=false
其他字符串=true

强制数据类型转换


①调用被转换数据类型的toString()方法
只适用于Number Boolean
var a=a.toString();


②调用String()函数
将被转换的数据作为参数传递给函数
var a=String(a);
对于Number和Boolean底层调的仍是toString
对于Null和Undefined,则直接被转换为"Null"和"Undefined"


③+""
隐式的类型转换，由浏览器自动完成
实际上是在调用String()函数
var a=a+"";






Number 数值

基本数据类型


数字 整数 浮点数
转Boolean 正负数=true
0=false

浮点数运算的结果不一定精确，因为二进制不能精确表示1/10
*不能进行对精确度要求高的运算


Number.MAX_VALUE 最大值
转Boolean =true

1.7976931348623157e+308


Number.MIN_VALUE 最小值
转Boolean =true

最小的正值 5e-324


Infinity -Infinity 正 负无穷
转Boolean =true

使用Number表示的数值超过最大值，返回Infinity
使用Number表示的数值小于最小值，返回-Infinity


NaN 非数字
转Boolean =false

强制数据类型转换


①调用Number()函数
对于内容包含数值的字符串转换有局限
var a=Number(a);


②调用parseInt()函数 用于解析字符串
提取首个有效整数转换，条件满足于字符串的首位元素为数值
非字符串会被转换为字符串后再操作
var a=parseInt(a);


③调用parseFloat()函数 用于解析字符串
提取首个有效浮点数转换，条件满足于字符串的首位元素为数值
非字符串会被转换为字符串后再操作
var a=parseFloat(a);


④隐式的类型转换
原理和Number()函数一样
var a=a-0;
var a=a*1;
var a=a/1;


⑤隐式的类型转换
原理和Number()函数一样
var a=+a;

var a=!!a;
